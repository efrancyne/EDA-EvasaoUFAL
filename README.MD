
# Análise de Dados — UFAL em Números

Descrição
--------

Projeto de análise exploratória dos dados públicos "UFAL em Números". O objetivo é investigar padrões de evasão e permanência nos cursos da Universidade Federal de Alagoas (UFAL), comparar unidades, identificar cursos com maior taxa de evasão e fixar conhecimentos sobre bibliotecas e técnicas aplicadas à análise de dados.

Objetivos
---------

- Responder perguntas como:
	- Quais cursos apresentam maior taxa de evasão?
	- Há diferença de evasão entre unidades (campi)?
	- Quais fatores estão associados à permanência ou evasão?
- Encontrar outros padrões relevantes nos dados.
- Praticar e demonstrar uso de bibliotecas de análise (pandas, numpy, matplotlib, seaborn, scikit-learn, etc.).

Dados
-----

Os dados usados são os disponibilizados pelo projeto "UFAL em Números".

Metodologia
----------

- Limpeza e pré-processamento dos dados (tratamento de valores faltantes, padronização de colunas).
- Análises descritivas (tabelas, médias, taxas, distribuições).
- Visualizações (gráficos de linha, barras, mapas, heatmaps).
- Modelagem exploratória quando pertinente (por exemplo, classificação ou regressão para identificar fatores associados à evasão).

Como replicar
--------------

1. Clonar o repositório
2. Criar um ambiente virtual e instalar dependências (requirements.txt)
3. Executar os notebooks em notebooks/ na ordem desejada
